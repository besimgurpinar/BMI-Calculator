JavaScript BMI Calculation App
This is a simple web application that calculates Body Mass Index (BMI) using JavaScript.

How to Use
Clone this repository to your local machine:
bash

git clone https://github.com/your-username/js-bmi-calculator.git
Navigate to the project directory:
bash

cd js-bmi-calculator
Open index.html in your web browser.

Enter your weight in kilograms and height in meters in the input fields and click the "Calculate BMI" button.

The calculated BMI value will be displayed below the input fields.
